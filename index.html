Selected work

MODULE 01

ESP32-CAM Smart Home Security System

Designed and developed a smart home security system using ESP32-CAM, PIR HC-SR501, and RCWL-0516 sensors to detect human presence and motion. Integrated the system with the Telegram Bot API to send real-time security alerts, automatically capture images when motion was detected, and transmit the captured photos to users. Focused on reliable sensor integration, embedded programming, and real-time IoT communication to provide remote home monitoring.

MODULE 02

Smart Aquascape Monitoring and Control System

Designed and developed an IoT-based smart aquascape system consisting of embedded software and a web application. The system enables real-time monitoring and control of key aquarium parameters, including water temperature, pH levels, and automated fish feeding. Implemented seamless communication between the hardware and web platform to ensure reliable data transmission, remote monitoring, and efficient aquascape management.

MODULE 03

Web-Based Ticket Management System with ESP32-CAM Integration

Web-Based Ticket Management System with ESP32-CAM Integration Designed and developed a web-based ticket management system integrated with an ESP32-CAM ticket scanning device. Implemented real-time communication between the scanning hardware and the web application to validate tickets efficiently. Developed features for online ticket purchasing, ticket verification, and a management dashboard that enables owners to monitor ticket sales, transactions, and visitor data through a centralized interface.
